# Keeping You Awake Control

Control [KeepingYouAwake](https://keepingyouawake.app/) from Raycast — toggle, activate for a specific duration, or deactivate.

## Prerequisites

Install **KeepingYouAwake** (free, open-source):
[https://keepingyouawake.app/](https://keepingyouawake.app/)

## Commands

| Command | Default Alias | Description |
|---------|---------------|-------------|
| Toggle | `ka` | Toggle KeepingYouAwake on/off |
| Deactivate | `kd` | Deactivate KeepingYouAwake |
| Activate for Hours | `kh` | Activate for N hours (e.g. `kh 8`) |
| Activate for Minutes | `km` | Activate for N minutes (e.g. `km 30`) |

## Customizing Aliases

All command aliases can be changed in Raycast Settings → Extensions → Keeping You Awake Control.
